# contracts/legacy

Each legacy URL will have one contract file.

Contract format (per page):

- `url`
- `page_type`
- `status` (`keep`/`merge`/`301`/`kill`)
- `title`
- `meta_description`
- `canonical`
- `h1`
- `heading_inventory`
- `required_body_sections`
- `required_internal_links`
- `required_images_alt_text`
- `schema_expectations`
- `approved_deltas`
